# 平台封面：同一份資料，兩種對立的第一眼

CD4⁺ T 細胞 Perturb-seq 標的發現平台的封面，呈現兩入口架構的核心張力。

- **x 軸（研究者視角）**：context-specific 效應（活化後下游 DE 廣度，log）— 越右越新穎、機制潛力越高
- **y 軸（臨床醫師視角）**：臨床風險訊號數（泛效應 + 遺傳約束 LOEUF<0.35 + 劑量敏感 pLI≥0.9）— 越上越危險
- **衝突區（右上陰影）**：研究者最想要的高效應 context-specific 標的（VAV1、STAT3），正是醫師避雷清單上的高風險項目

核心訊息：**在生物學上舉足輕重與遺傳上不可安全干預，常是同一件事的兩面**。這正是兩入口存在的理由——同一份證據要用兩種風險語言講。

資料來源：`kinetics_avoid/target_master_table.csv`（96 context-specific 標的，gnomAD 97% 覆蓋）。
