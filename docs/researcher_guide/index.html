如何把這個平台當成
研究工具來讀與用

這是給計算生物、免疫學與藥物探索研究者的導覽:平台在科學上回答什麼、每個輸出怎麼判讀、如何呼叫 API、如何讀圖,以及每個宣稱的權威依據在哪。它是**決策支援層**——消費已發表的 GWT Perturb-seq DE 結果 [1],不重算 DE。

決策支援層,非新方法 consume ≠ recompute 研究用,非臨床建議

30 秒定位

這個平台在科學上回答什麼

它把一個在原代人類 CD4⁺ T 細胞上執行的全基因體 CRISPRi Perturb-seq 篩選 [1] 的差異表現(DE)結果,轉為每「標的 × 條件」一列的標靶卡片,並輸出可辯護、可追溯的 advance validate watchlist deprioritize 決策。核心價值不在演算法,而在:把統計等級、穩健性、敲低因果性、可成藥性、安全與遺傳學證據整合成一個**保守、誠實、可稽核**的優先序。

關鍵前提

平台不重算 DE / pseudobulk / NTC 基線——那些是上游已驗證的計算。它加的是判讀與決策層。方法學全貌見 docs/technical_methods.md。

核心判讀

怎麼讀一張標靶卡片

一張卡片是「一個標的在一個條件下」的完整證據摘要。判讀時依序看四件事:

① 統計證據等級 1–4

statistical_evidence_grade:建立在上游 keep_for_DE 之上的可重現性閘(最小細胞數、跨 donor/guide 相關、脫靶旗)。缺 guide 資料封頂 2。

② 敲低因果閘 kd_status

confirmed(可因果判讀)/ weak / not_measurable(NTC 基線 ≤0.001)/ not_assessed(從未測量,不懲罰)。標的自身未確認敲低,下游 DE 不可因果歸因。

③ 穩健性

crossdonor_correlation 與 crossguide_correlation:效應在不同捐贈者與不同 gRNA 間是否重現。缺值顯示 caveat,不補 0。

④ 決策 + 封頂原因

readiness_call + score_cap_reason + next_step。永遠先看封頂原因,它解釋為何某列沒到更高等級。

判讀守則

unknown ≠ 0:未建置的證據領域是明確「未知」,不是負分。低 grade 搭配「未被 deprioritize」的 readiness 本身即有資訊——兩者回答不同問題。

決策邏輯

就緒度引擎與紅旗覆蓋

12 個證據領域 → R0–R5 階梯 → 決策。關鍵在紅旗覆蓋:不論下游統計多強,命中即封頂——這是平台把「訊號強」與「好標的」分開的機制。

  • essential_gene → watchlist必需基因;另不得捏造 safety 分數
  • broad_effect → watchlist廣泛效應(239 基因清單:染色質/轉錄機器)
  • high_offtarget → watchlistofftarget_flag(TSS 10 kb 內顯著下調)
  • kd_not_measurable → watchlist敲低無法評估
  • uncertain_direction → validateontarget_significant 為否 / 方向不明
  • batch_confounded → validatebatch_sensitivity_flag = sensitive
  • kd_weak → validate敲低訊號未達確認門檻

端點 GET /api/readiness/{dataset_id}。完整領域與門檻見 docs/technical_methods.md §3.5。

概念層

免疫概念模組 M01–M20

平台把每個標的/樣本投影到 20 個 CD4 T 細胞免疫概念(concept-bottleneck layer,COMPASS-analog),以 seed-gene overlap 計分(可手算稽核、無黑箱)。涵蓋 TCR 訊號、共刺激/檢查點、細胞激素軸、Th 分化、代謝、記憶/耗竭、細胞週期。

關鍵不變量

概念分數是描述性的,永不餵 readiness_call / _stage()。它只回答「這個標的在哪些免疫概念上有訊號」,不進評分。

端點 GET /api/modules/{dataset_id}、POST /api/individual-concept-profile。逐模組 seed genes 與生物學意義見 docs/concept_dictionary.md。

限制 · 引用

限制與如何引用

明確 descope(非疏漏,已記錄):signed module scoring(DE 僅 up/down 計數、無 per-gene 方向)· SCEPTRE(誠實外部 hook 而非天真重寫)· pertpy/Mixscape(scikit-learn 替代)· 細胞層級真實資料(1.7 TiB 委外執行,僅對合成 fixture 驗證)· 情境專一性為 heuristic 非交互作用檢定。

CRISPRi ≠ 藥理學

體外 CD4 情境的強訊號是線索,不是療效保證;安全與可成藥性永遠拉回人體證據。本平台輸出為研究用標靶優先排序,非臨床或用藥建議。

引用:Zhu R., Dann E. et al. (2025) Genome-scale perturb-seq in primary human CD4⁺ T cells… bioRxiv 10.64898/2025.12.23.696273 [1]。
